# Sprint_5

## Автотесты для Stellar Burgers

Проект содержит автоматизированные тесты для приложения **Stellar Burgers**. Тесты написаны на **Selenium3**



### Описание файлов

####  **tests/**
- **`conftest.py`** - Фикстуры и общая конфигурация pytest
- **`test_registration.py`** - Тесты регистрации пользователя (2 теста)
- **`test_login.py`** - Тесты авторизации (4 теста)
- **`test_navigation.py`** - Тесты навигации по сайту (4 теста)
- **`test_constructor.py`** - Тесты конструктора бургеров (3 теста)

####  **pages/** (Page Object Pattern)
- **`__init__.py`** - Инициализация пакета
- **`main_page.py`** - Методы для работы с главной страницей
- **`login_page.py`** - Методы для страницы авторизации
- **`register_page.py`** - Методы для страницы регистрации
- **`profile_page.py`** - Методы для личного кабинета
- **`forgotpass_page.py`** - Методы для восстановления пароля

####  **Основные файлы**
- **`locators.py`** - Локаторы элементов (селекторы, XPath, CSS)
- **`generators.py`** - Генерация тестовых данных (email, пароли, имена)
- **`requirements.txt`** - Список зависимостей Python
